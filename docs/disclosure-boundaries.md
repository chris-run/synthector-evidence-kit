# Disclosure Boundaries

This public repository is deliberately limited to synthetic examples and local verification code.

It excludes:

- Proprietary redaction and minimization engine code.
- Detector, fusion, ranking, or redaction rules.
- Production API URLs.
- Keys, tenant identifiers, credentials, and operational configuration.
- Real JWS receipts.
- Personal data.
- Customer records, transcripts, notes, or support artifacts.

The sample inputs, outputs, and receipts are illustrative. They are intended to help technical reviewers understand the public evidence shape without expanding the disclosure surface of the private system.
